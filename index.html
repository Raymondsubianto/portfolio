3rd Portfolio



Pelayanan Pelanggan

Pelayanan Pelanggan is an Information System website for service customer about service request and complain that I developed during internship in UP3 Klaten. Beside that, this web also save the power used by customer, recording switching, and blackout announcement. This web implement the Create, Read, and Update (CRUD). For programming language, I use PHP for query data from MySQL Database and for serve data using HTML and CSS. For web server, this web use Apache that provisioned in XAMPP software. So, this web can run in localhost.

This web have 2 roles, such as admin and user. Each role have different access of each pages. Admin have access to see all data from any user, but the user only see their own data. Not only that, is some pages like blackout announcement, user only could see the blackout announcement, but the admin could give the announcement and delete it. The other feature of this website is user could create a PDF file for the complaints and service requests, so they don't need to write it manually.

Documented in: https://github.com/Raymondsubianto/layanan-pelanggan.git
